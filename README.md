# Multi-vendor LLM prompt schema (NMRG Internet-Draft)

This is the working area for the Internet-Draft “Framework for Normalizing Multi-Vendor Network Inputs for LLM-Assisted Network Management”, aimed at the IRTF [Network Management Research Group (NMRG)](https://datatracker.ietf.org/rg/nmrg/about/).
